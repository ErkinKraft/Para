# Para - Изменения (V1.1)

## V1.1

- Исправлен ложный статус "Подключено" - теперь подтверждение соединения происходит через явный handshake.
- Добавлены: отмена/отключение по кнопке "Назад" (обе стороны корректно закрывают соединение).
- Пароль на подключение: принимающая сторона может требовать пароль; отправитель отправляет пароль при подключении.
- Локальное обнаружение (LAN discovery): принимающий узел отвечает на запросы, отправитель может найти и автозаполнить IP/порт.
- Системный трей: свёртывание в трей, показ/скрытие консоли, восстановление окна, уведомления при разрыве.
- Автопереключение раскладки на английскую перед началом отправки/приёма.
- Обновлён интерфейс (тёмная тема, аккуратные поля/кнопки, иконки показа пароля).
- Кнопки "Показать/скрыть пароль" рядом с полями пароля.
- Кнопка "Справка" с инструкциями и разделом "О программе".
- Поле порта у отправителя по умолчанию пустое ("Введите порт").

## Известные моменты

- Для работы обнаружения по локальной сети необходимо разрешить UDP на порту 53555 и включить настройку "Разрешить обнаружение…" на принимающем.
- Для эмуляции клавиш на принимающей стороне может требоваться запуск "От имени администратора" и/или ручного переключения языка на ENG.
